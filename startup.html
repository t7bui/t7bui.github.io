Usability Testing

What need to improve:

Introduction/Exterior Tour:

  • CONFUSION with White Arrow - Many users thought the White Arrow would lead them inside the house
  • BIG ISSUE with Drag Animation - Many users thought it prompts them to drag the screen (see video below)

Interior Tour:

  • NOT CLEAR with Home Button - users did not know how to exit tour using Home Button (see video below)
  • NOT VISIBLE with Garden Button - users did not realize there is a Garden Button in the corner (see video below)

  • Interesting Observation - when stuck, users try to exit the house by tapping outside the door (see video below)

Stuart Collection Page:

  • Suggestions - Add sculptures' locations and add option to enlarge photos
  • Interesting Observation - everyone's FIRST INSTINCT is to drag the page up
  • BIG ISSUE with Gallery Flow - User 3 notices the gallery flow reset when clicking on

What we changed:

Introduction/Exterior Tour:

  • White Arrow - Have the arrow leads users inside the house (no zoom foward)
  • Drag Animation - Change finger to arrows (might indicate less action)
  • Wording of Instruction - Move action words to the end

Interior Tour:

  • Home Button - Change "Home" under icon to "Exit Tour"
  • Garden Button - Remove this button, and have mascot ask at the door "Want to go outside?"; more natural this way

Stuart Collection Page:

  • Add scrulptures' locations and also have option to see more photos based on suggestions
  • Extend the page a bit longer, so that would allow users to drag up when they enter the page
  • Fix Gallery Flow!

Reflections

What I learned

  • First-time team management experience taught me valuable lessons in effective communication, planning, and task delegation.
  • Active involvement in interview processes, UI design, and usability testing expanded my skills in interviewing, receiving feedback, and utilizing Figma for design enhancements.

If we have more time

  • Unresolved issue: Usability test revealed confusion with the dragging animation for exterior tour. Given more time, we would enhance the interaction for better user intuitiveness.
  • Consistency suggestion: Consider incorporating virtual tour format for Stuart Collection Gallery to align with Fallen Star and meet Prof's feedback.
  • Enhanced usability: Enable dragging while Yellow Button is clicked to prevent information text cutoff confusion during exterior tour.
  • Text summarization opportunity: Given additional time, we could have summarized the Stuart Collection Gallery texts in our own words instead of using direct website excerpts.
  • Navigation optimization: More testing needed to determine the most effective method for guiding users' attention to the arrow navigation in the interior tour.
  • Limited kiosk testing: Due to time constraints, the physical kiosk iterations were based on member feedback rather than user testing.
